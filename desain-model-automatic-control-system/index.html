üéØ Tujuan dan Fungsi Sistem

Sistem ini dirancang sebagai Automatic Control System yang secara real-time memantau kondisi vital ruangan meliputi suhu, kelembapan udara, dan konsentrasi gas berbahaya. Ketika salah satu parameter melampaui ambang batas (setpoint), sistem akan otomatis mengaktifkan kipas untuk sirkulasi udara dan buzzer sebagai alarm peringatan tanpa memerlukan intervensi manual.

üîß Komponen Sistem Berdasarkan Fungsi

üì° 1. BAGIAN INPUT (Sensing Layer)

Lapisan ini bertugas mengumpulkan data lingkungan secara real-time

üå°Ô∏è Sensor DHT11 / DHT22

Fungsi: Mengukur suhu dan kelembapan udara secara simultan

Tipe: Sensor Digital

Output: Data digital via protokol 1-wire

DHT11: Suhu 0-50¬∞C (¬±2¬∞C), Kelembapan 20-80% (¬±5%)
DHT22: Suhu -40-80¬∞C (¬±0.5¬∞C), Kelembapan 0-100% (¬±2%) - Lebih Akurat

üí® Sensor Gas MQ-2

Fungsi: Mendeteksi gas mudah terbakar dan asap

Gas Terdeteksi: LPG, Propana, Metana, Hidrogen, Asap

Output: Sinyal analog (0-1023)

Range: 300-10,000 ppm
Waktu Pemanasan: ~20 detik

üß† 2. BAGIAN PROSES (Control Layer)

Unit pemrosesan yang menjalankan logika pengambilan keputusan

‚ö° Arduino Uno R3

Fungsi: Mikrokontroler utama sebagai otak sistem

Prosesor: ATmega328P

Tugas: Membaca sensor, membandingkan dengan threshold, mengontrol aktuator

Digital I/O: 14 pin (6 PWM)
Analog Input: 6 pin (10-bit ADC)
Tegangan: 5V DC

üì∫ LCD 16x2 + Modul I2C (Opsional)

Fungsi: Menampilkan data real-time

Informasi Ditampilkan: Suhu (¬∞C), Kelembapan (%), Level Gas (ppm)

Modul I2C: Menyederhanakan koneksi (hanya 2 pin: SDA & SCL)

Display: 16 karakter x 2 baris
Backlight: Biru/Hijau

‚öôÔ∏è 3. BAGIAN OUTPUT (Actuator Layer)

Perangkat yang mengeksekusi perintah kontroler

üåÄ Kipas DC 12V

Fungsi: Sirkulasi udara untuk menurunkan suhu dan membuang gas berbahaya

Kontrol: Via Modul Relay (ON/OFF)

Tegangan: 12V DC
Arus: ~0.1-0.3A

üîä Buzzer Aktif 5V

Fungsi: Alarm peringatan audio saat kondisi berbahaya

Tipe: Active Buzzer (menghasilkan suara sendiri)

Kontrol: Langsung dari pin digital Arduino

Tegangan: 5V DC
Frekuensi: ~2-4 kHz

üîå Modul Relay 1 Channel 5V

Fungsi: Saklar elektronik untuk kipas 12V

Peran: Jembatan antara Arduino (5V) dan Kipas (12V)

Prinsip: Sinyal LOW/HIGH dari Arduino mengontrol relay untuk menyambung/memutus arus kipas

Coil: 5V DC (dari Arduino)
Contact: 250V AC / 30V DC, 10A

‚öôÔ∏è Prinsip Kerja dan Alur Logika Sistem

1Ô∏è‚É£ Inisialisasi Sistem

Saat power ON, Arduino melakukan setup:

  • Inisialisasi sensor DHT dan MQ-2
  • Setup LCD I2C (alamat 0x27 atau 0x3F)
  • Konfigurasi pin relay dan buzzer sebagai OUTPUT
  • Set nilai threshold (batas aman)
‚Üì

2Ô∏è‚É£ Pembacaan Data Sensor (Loop Kontinyu)

Arduino membaca sensor setiap interval waktu (misal: setiap 2 detik)

  • Baca suhu (¬∞C) dan kelembapan (%) dari DHT11/22
  • Baca nilai analog gas dari MQ-2 (0-1023), konversi ke ppm
  • Validasi data (cek error sensor)
‚Üì

3Ô∏è‚É£ Tampilkan Data ke LCD

Update display setiap cycle:

  • Baris 1: "Temp:28C Hum:65%"
  • Baris 2: "Gas:250ppm"
‚Üì

4Ô∏è‚É£ Evaluasi Kondisi (Decision Making)

Arduino membandingkan nilai sensor dengan threshold:

IF (suhu > 30¬∞C) OR (kelembapan > 70%) OR (gas > 300 ppm) THEN kondisi = TIDAK AMAN ‚ö†Ô∏è ELSE kondisi = AMAN ‚úÖ
‚Üì

5Ô∏è‚É£ Aksi Kontrol (Eksekusi)

Jika TIDAK AMAN:

  • ‚úÖ Arduino kirim sinyal HIGH ke Relay ‚Üí Kipas ON
  • ‚úÖ Arduino kirim sinyal HIGH ke Buzzer ‚Üí Alarm ON
  • ‚úÖ LCD menampilkan "‚ö†Ô∏è PERINGATAN!"

Jika AMAN:

  • ‚úÖ Arduino kirim sinyal LOW ke Relay ‚Üí Kipas OFF
  • ‚úÖ Arduino kirim sinyal LOW ke Buzzer ‚Üí Alarm OFF
  • ‚úÖ LCD menampilkan "‚úì Normal"
‚Üì

6Ô∏è‚É£ Delay dan Ulangi

Sistem menunggu interval waktu (misal: 2000ms), kemudian kembali ke langkah 2

üìö Kesimpulan

Sistem Monitoring dan Kendali Otomatis Kondisi Ruangan ini merupakan implementasi praktis dari Closed-Loop Automatic Control System yang menggabungkan teknologi sensing, processing, dan actuation. Dengan kemampuan monitoring real-time dan respons otomatis terhadap kondisi berbahaya, sistem ini efektif meningkatkan keselamatan dan kenyamanan ruangan. Platform Arduino memberikan fleksibilitas tinggi untuk pengembangan lanjutan seperti IoT, machine learning, dan sistem kontrol yang lebih kompleks.
